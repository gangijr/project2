Some example curl command that you can run on this project are :

curl https://project2.jgangi.me:8443/players

Will get a list of all players currently in the data base

curl https://project2.jgangi.me:8443/players/?

When ‘?’ is an integer value that matches to a ID it will get that particular player

curl --header 'Content-Type: application/json' \ --data '{“fullName“: “name”, screenName" : “test”}' \ https://project2.jgangi.me:8443/players

This will add a player to the database and automatically assigns a ID to it based on table size

curl --header 'Content-Type: application/json' \ --request PATCH \ --data '{ screenName" : “testing_patch”}}' \ https://project2.jgangi.me:8443/players/2

This would change the screenName of the player with ID 2 to ‘testing_patch’

curl --request DELETE \ https://project2.jgangi.me:8443/players/2

This command would delete player2 from the database (soft delete)
